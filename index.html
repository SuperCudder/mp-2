Country Guesser

Minigame for travelers

Welcome

This is a small scale React based mini-project challenging users to identify countres given a 360-degree panoramic street view image. Built using TypeScript and the Mapillary API, this game calls random goelocated photos from around the world and presents users with multiple choice questions. Utilizing caching for smooth gameplay, real time score tracking and a detailed viewer experience. Also featuring custom event handling for component communication, and efficient API data management.
